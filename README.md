# HeisLabBiggest  

## Beskrivelse  

Dette prosjektet implementerer et heissystem i faget TTK4145. Sanntidssystemet er skrevt i Go.

## Hvordan kjøre  
### 1. Start simulator,  Åpen en terminal:   ```./SimElevatorServer --port 15657 ```  
### 2. Build programmet, Åpen en terminal: ```go build -o heis main.go```
### 3. Kjør programmet,  Åpen en ny terminal:  ```./heis -port 15657 ```, Porten må være lik begge steder.
